# CSCI 8960 Thesis Equivalent Project

## Project Focus

The project explores how to analyze and optimize 3D USDZ files for AR performance on iOS devices.

The main goal is to study how factors such as file size, polygon count, object count, materials, textures, and device capability affect whether a USDZ file loads smoothly, performs well, or becomes unstable.

## Current Work

- Testing USDZ files on different iPhone models
- Auditing 3D models using Blender scripts
- Collecting model metrics such as vertices, faces, triangles, materials, and textures
- Experimenting with optimization workflows such as decimation and normal baking
- Building toward a process for identifying performance issues in AR-ready 3D assets

## Tools Used

- Blender
- Python
- USDZ / AR Quick Look
- iPhone 16 Pro
- GitHub for version control and documentation
